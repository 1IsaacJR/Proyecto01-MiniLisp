En este proyecto se presentan un total de 7 archivos que permiten la ejecucion completa del lenguaje, la composicion del programa esta dada de la siguiente forma: 
            1.ASA.hs 
            2.ASAValues.hs
            3.Desugar.hs
            4.SASA.hs
            5.Interprete.hs
            6.Main.hs
            7.Lexer.x (Lexer.hs)
            8.Parser.y (Parser.hs)
1. Denota las reglas de Sintaxis Abstracta propuestas en la formalizacion del lenguaje
2. Presentamos los valores finales de las reglas de Sintxis Abstracta que nos permitiran o proporcionaran los valores finales de ejecucion en el lenguaje
3. Se inicializa la desazucarizacion de las reglas de semantica propuestas en el archivo, proporcionandonos el efoque de simplicidad, entendimiento y facilidad de las reglas de Sintaxis Abstracta para la persona que utilice el lenguaje
4. Proporcionamos las reglas de Azucar de Sintaxis Abstracta, que seran utilizadas dentro de Desugar.hs, Interprete.hs. Nos permitiran pasar de la sintaxis con azucar a un nucleo de sintaxis abstracta para la facilidada de la operatividada con el lenguaje
5. Se implementan las reglas de semantica operacional propuestas para la interaccion con el lenguaje extendido
6. Se presentan casos de prueba, asi como tambien parte de las facilidades del menu interactivo con el usuario
7. Se hace uso de la herramienta Axel para implementar la gramatica de nuestro lenguaje de programacion
8. Se hace uso de la herramienta Happy para la implementacion de las expresiones regulares dentro de la formalizacion del lenguaje.
